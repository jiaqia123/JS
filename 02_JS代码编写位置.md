# `JS`代码编写位置

## 一、内部编写

```html
<body>
  <script>
    // 弹窗
    alert('你好JS');
  </script>
</body>
```



## 二、外部编写

```html
<body>
  <!-- src外部JS路径 -->
  <script src="./JS/lode.js"></script>
</body>
```



## 三、内联样式

```html
<body>
  <!-- 弹窗 -->
  <button onclick="alert('你好')">你好呀</button>
</body>
```



**注意**：

- 书写位置尽量写到文档末尾`</body>`前面
- 外部`JS`标签中间不要写代码，会被忽略
